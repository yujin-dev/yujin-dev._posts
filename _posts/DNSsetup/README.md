# DNS 설정
name server를 명시하여 Cloud DNS에 등록한 주소 IP를 확인할 수 있다.
```
$ nslookup dns.xxx ns-cloud-d1.googledomains.com
Server:		ns-cloud-d1.googledomains.com
Address:	216.239.32.109#53

Name:	dns.xxx
Address: 34.xx.xxx.xxx
```
